# State-Frequency-Memory
- 状态频率记忆网络（State Frequency Memory，SFM）是一种新颖的循环网络，它使用离散傅里叶变换来分解记忆单元的隐藏状态，并从过去的市场数据中捕捉多频率交易模式，从而进行股票价格预测。
- 论文：Stock Price Prediction via Discovering Multi-Frequency Trading Patterns. [http://www.eecs.ucf.edu/~gqi/publications/kdd2017_stock.pdf.](http://www.eecs.ucf.edu/~gqi/publications/kdd2017_stock.pdf)
